# One run folder = one recording session

Drop exactly **one** primary video into this folder (for example
`session.mp4`). The folder name (`example_run`) is the run id -- rename it to
something meaningful for your study, e.g. `session_01`.

Optionally add a `run.yaml` here to record participant labels and the condition
for this session:

```yaml
participants: [P0, P1]
condition: baseline
```

Delete this README once you add a real video.
